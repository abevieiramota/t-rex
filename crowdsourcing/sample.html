Is The Fact In The Paragraph?

Instructions

Overview

In this task, contributors are asked to check if facts are true or false according to the displayed paragraph.  

Annotators should keep in mind the following procedures: 

  1. If a fact is true but not necessarily expressed in the paragraph it must be considered as false. 
  2. A fact can be labeled only as True if the sentences mentioned in a paragraph implies the fact with 100% confidence.


    Example:
    		label
    		why ?
    X is the son of A and B 
    A   <spouse>   B 

    		False
    		many parents of similar children aren't in a marriage relationship, so should be labeled as False.
    X and Y   are the children of  of A and B
    X  <brother of>  Y 
    		True
    		Children of the same parents are brothers
    X  is a metal guitarist playing for the band Y
    Y <has genre> Metal 
    		True
    		A band who has a metal guitarist is a metal band.

Steps

  1. Read the paragraph carefully.
  2. Read each one of the candidate facts that are enumerated below each paragraph.
  3. Check the True or False radio buttons below each fact. The assessment must be made only according to the information contained in the paragraph.
  4. Ambiguous cases are left to annotators agreement.
